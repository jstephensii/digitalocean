install zsh - DON'T use brew
-- sudo apt-get install zsh

switch from bash to zsh (Option #1)
-- set zsh to default - chsh -s $(which zsh)

NOTE: If you see a "chsh: PAM: Authentication failure" then you are probably not running Ubuntu, and will need to switch from Bash to ZSH using another method.
- to check your OS version run: "cat /etc/os-release"

switch from bash to zsh (Option #2)
-- sudo vi /etc/passwd
- find the line with "[username]:x:1000:1000:[username]:/home/[username]:/bin/bash"
- replace the 'bash' with 'zsh'
-- [username]:x:1000:1000:[username]:/home/[username]:/bin/zsh

verify that zsh is now running
- logout of server, login, and verify zsh is default shell
-- echo $SHELL
